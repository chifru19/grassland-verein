# Grassland Verein e.V. Berlin

Digital platform for **Deutsch-Kamerunischer Kulturverein Grassland e.V. Berlin**. This project was built using React and is deployed via GitHub Pages.

## ⚙️ Available Scripts

In the project directory, you can run:

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### `npm run build`
Builds the app for production to the `build` folder.

### `npm run deploy`
Deploys the current production build to GitHub Pages.
